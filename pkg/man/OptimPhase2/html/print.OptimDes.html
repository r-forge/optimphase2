Print Objects Output By OptimDes

Description

Print an object output by OptimDes.

Usage

## S3 method for class 'OptimDes':
print(x, dig = 3, all=FALSE, condPow=F,
                     CMadj=F, ...)

Arguments

x Object output by OptimDes.
dig Number of digits printed.
all If TRUE, results are printed for all total sample sizes evaluated by OptimDes.
condPow Conditional power at the test boundary C1 under H1 is reported when condPow=T
CMadj If true, the sample sizes and times are adjusted by the ratio of the exact binomial to asymptotic normal sample size for the single stage design, as in Case and Morgan 2003. Proportional adjustment of times and sample sizes are made even if the accrual rates are not constant. Default is false.
... Optional print arguments, see printt.default.

Author(s)

Bo Huang <huang@stat.wisc.edu> and Neal Thomas

References

Simon R. (1989) Optimal two-stage designs for phase II clinical trials. Controlled Clinical Trials, 10, 1–10.
